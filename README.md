- # Privacy Policy
การเก็บข้อมูลของบอท Furina
`-` สำหรับข้อมูลถาวรที่อยู่ในDatabase ( จะถูกลบอัตโนมัติเมื่อบอทออกจากเซิฟ )
1. ID และชื่อเจ้าของเซิฟเวอร์ , ชื่อเซิฟเวอร์ , ไอดีเซิฟเวอร์
2. ไอดีบทบาทที่คุณตั้งค่า ระบบมอบบทบาทอัตโนมัติ และระบบอื่นๆในอนาคต
3. ไอดี ช่องแชทที่คุณเปิดใช้งานระบบ ต่างๆของฟูรินะ เช่น ระบบห้องเกมทั้งหมด,ระบบlog-messages
`-` ทางเราให้ความสำคัญกับข้อมูลส่วนบุคคลของท่าน หากไม่พึงประสงค์จะให้เก็บสามารถทัก DM มาขอลบได้ครับ ส่วนข้อมูลอื่นๆ ณ ตอนนี้ยังไม่มีจัดเก็บ
